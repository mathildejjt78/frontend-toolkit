
# Boilerplate

- HTML : `index.html`
- CSS : `style.css`
- JS : `script.js`
- SCSS: `main.scss`



```bash
npm init
```
    

## Installation


### - Vite

#### - Ajoute : 
- `node_modules`
- `package-lock.json`


### - Jest

#### - Ajoute : 
- `__tests__` + `sum.test.js`.
- `sum.js`


` npm run test `

### - ESLint

#### - Ajoute : 
- `.eslintrc.json` or `eslint --init`



### - Husky

#### - Ajoute automatiquement : 
- `pre-commit` 
- lance `npm test` avant chaques commit


### - Sass

#### - Ajoute automatiquement : 
- `pre-commit` 
- lance `npm test` avant chaques commit
